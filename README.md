## Dashboard

dashboard is private CMS for personal project management system using Bootstrap, Sass, Javascript, Php laravel 8.

## Packages Used

-   Install `composer require laravel/breeze --dev`
-   Install `composer require laravel/horizon`
-   Install `composer require predis/predis`
-   Install `composer require barryvdh/laravel-debugbar --dev`
-   Install `npm install sweetalert2`
-   Install `composer require appslabke/lara-izitoast`
